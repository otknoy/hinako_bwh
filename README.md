# Hinako API 

## Requirements

* Python 2.7
  - pyfirmata
  - SPARQLWrapper
  - bottle

	You can install these modules using pip.

## How to use

### Hinako API

* Please change some port settings in hinako.py:
  - Arduino port number
  - Bust servo motor port number
  - Waist DC motor port number
  - Hip servo motor port number

    $ cd hinako_api
    $ ./hinako_api.py


### Measurements API

    $ cd hinako_api
    $ ./hinako_api.py
